# VERNY 주문서

VERNY(베르니 · (주)베러스) 프리미엄 비즈니스 양말 **주문서 웹앱**. 단일 `index.html`, 서버 없이 GitHub Pages로 배포. 링크만 보내면 손님이 입력 → 실시간 합계 → 주문 시 담당자 이메일 발송 + 입금계좌 안내.

라이브: https://heegeun84-ai.github.io/verny-order/

## 기능
- 상품 선택: 선물세트(3족·블랙/딥네이비/차콜) · 낱개(색상별) · 고급 쇼핑백
- **수량별 자동 가격**: 세트+낱개 합계 **10개 이상 → 대량가(03) + 무료배송**, 미만 → 채널가(02) + 배송비 3,000원
- 실시간 합계 · 티어 전환 안내(넛지)
- 주문자/수취인/배송지, 증빙(현금영수증·세금계산서)
- 주문 접수 → 담당자 이메일 자동 발송 + 주문번호 + 입금계좌 안내(복사)

## 설정 (index.html 상단 `<script>`)
| 항목 | 위치 | 비고 |
|---|---|---|
| 주문 알림 이메일 | `WEB3FORMS_ACCESS_KEY` | web3forms.com 발급 키. 미입력 시 미리보기 모드(이메일 미발송) |
| 가격표 | `PRICE.ch` / `PRICE.bulk` | 채널가(02) / 대량가(03) |
| 대량 기준 수량 | `BULK_MIN` | 기본 10 (세트+낱개 합계) |
| 배송비 | `SHIP_FEE` | 기본 3,000 |
| 입금계좌 | `BANK` | 신한 100-036-765551 (주)베러스 |

## 주문 이메일 연결 (Web3Forms · 무료)
1. https://web3forms.com 접속 → 주문 받을 **이메일 입력** → 메일로 온 **Access Key** 복사
2. `index.html`의 `WEB3FORMS_ACCESS_KEY = "___"` 자리에 붙여넣기 → 커밋/푸시
3. 이후 주문 접수 시 해당 이메일로 주문 내역 자동 발송 (무제한 무료, 이메일 주소는 소스에 노출되지 않음)
